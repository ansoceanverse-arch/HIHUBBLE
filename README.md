# Hi-HUBBLE

**Hi-HUBBLE** is a next-generation social media prototype designed as a Progressive Web App (PWA). It provides a full suite of modern social networking features seamlessly bridging the gap between web and mobile experiences.


## Tech Stack

| Category | Technologies |
| :--- | :--- |
| **Frontend** | Vite, HTML/CSS/JS (PWA configuration with `vite-plugin-pwa`) |
| **Backend** | Node.js, Express.js |
| **Database / BaaS** | Supabase |
| **Authentication** | JWT, bcryptjs |
| **Communications** | Twilio (SMS), Nodemailer (Email) |
